This document provides a comprehensive technical analysis of the components required for the assembly of self-contained computational systems. Anomaly Catalog: Component Analysis

The following is a high-level summary of each core anomaly. Click on a header to expand its summary. For detailed technical analysis, click on any highlighted term within the summary to access the relevant deep-dive documentation.

Anomaly-01: Central Processing Unit (CPU)

The CPU is the system's brain, executing instructions via a Fetch-Decode-Execute cycle. Its performance is a balance of IPC and Clock Speed. Modern CPUs use multiple Cores and Threads for parallelism, with some employing a hybrid architecture of Performance and Efficiency cores. A large CPU Cache is critical for reducing latency to main memory, especially in gaming.

Anomaly-02: Graphics Processing Unit (GPU)

The GPU is a massively parallel processor optimized for high-throughput tasks like rendering. Its architecture is based on the SIMD model, allowing it to perform the same operation on thousands of data points (pixels) simultaneously. It contains specialized hardware like RT Cores for ray tracing and Tensor Cores for AI, which power upscaling technologies like DLSS. Performance is heavily influenced by its dedicated VRAM capacity and bandwidth.

Anomaly-03: Motherboard

The motherboard is the system's central nervous system, connecting all components. Its most critical feature is the Voltage Regulator Module (VRM), which provides clean, stable power to the CPU. The motherboard's Chipset dictates its feature set, including connectivity options and support for overclocking. It also allocates PCIe lanes for communication between the CPU, GPU, and other peripherals.

Anomaly-04: Random Access Memory (RAM)

RAM is the system's volatile, high-speed workspace. Performance is a balance between capacity (GB) and speed (MT/s). True responsiveness is determined by its Absolute Latency, a calculation involving both speed and timings (like CL). Modern DDR5 memory offers significant architectural advantages over DDR4. To achieve advertised speeds, one must enable XMP or EXPO profiles in the BIOS.

Anomaly-05: Storage

Storage provides long-term, non-volatile memory. The hierarchy ranges from slow, high-capacity **HDDs** to fast **SATA SSDs**, up to the fastest **NVMe SSDs**. The performance difference is due to the interface (SATA/AHCI vs. PCIe/NVMe). SSDs themselves are built from different types of NAND flash (e.g., TLC, QLC), which determines their cost, performance, and endurance.

Anomaly-06: Power Supply Unit (PSU)

The PSU converts AC wall power to DC power for the system. Quality is paramount. Key metrics include wattage and the 80 PLUS efficiency rating. A good PSU features high-quality internal components, especially Japanese capacitors, and a full suite of protection circuits to ensure system safety and stability.

Anomaly-07: Thermal Management

Cooling is essential for performance and longevity. Air Coolers are reliable and cost-effective. All-In-One (AIO) Liquid Coolers offer superior thermal capacity for high-end CPUs. For ultimate performance, enthusiasts use Custom Liquid Cooling. All methods rely on effective Case Airflow to exhaust heat from the chassis.
